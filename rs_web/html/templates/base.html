• Query

Visualization and query interface of perception logs. Base queries commands:

  • object(Ob,Descr) - > shows all percieved objects from the belief state given a description
  • scenes(Sc,Descr) - > shows all scenes processesed by RoboSherlock given a description
  • hypotheses(objID) - > shows all hypotheses given a description

Documentation of the description language is available on the project website
